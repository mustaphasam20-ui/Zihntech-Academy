## 🌱 A beginner project with a long-term vision

This repository contains my **Zihntech Academy Landing Page**, one of the earliest projects in my frontend development journey.

More than just a landing page, this project represents a decision I made: **to stop only consuming technology and start building with it**.

As a beginner developer, I wanted to create something that demonstrates:

* 🧱 **Structured HTML**
* 🎨 **Responsive CSS**
* ⚡ **Basic JavaScript interactions**
* 📁 **Clean project organization**
* 📈 **The discipline of learning in public**

---

## 💡 Why this project matters to me

Every experienced developer once had a first project.

**This is mine.**

I understand that the code may not be perfect, the architecture may be simple, and the design choices may evolve over time. But this repository captures something valuable that cannot be copied from a tutorial: **the courage to begin**.

I am intentionally documenting my progress so that future versions of this project can show how much I have grown as a developer.

---

## 🧭 Project Overview

The goal of this project is to build a modern landing page for **Zihntech Academy** using core frontend technologies without relying heavily on frameworks.

### ✨ Features

* 📱 Responsive layout
* 🦸 Hero section
* 🧭 Navigation structure
* 🎯 Styled components with CSS
* ⚙️ Interactive behavior with JavaScript
* 🗂️ Organized asset management
