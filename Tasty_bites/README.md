## 🌱 Introduction

**Tasty Bites** is a responsive restaurant landing page created as part of my frontend development practice.

While the project is centered around food and hospitality, the real goal was much deeper: **to learn how professional websites are structured, styled, and made interactive using core web technologies**.

This repository represents one of the stages in my journey from **curious beginner to confident frontend developer**.

Instead of only watching tutorials, I challenged myself to build something that feels like a real business website — complete with a hero section, navigation, menu showcase, gallery, and contact experience.

---

## 💡 Why I Built This Project

Every developer has projects that mark important turning points in their learning journey.

**Tasty Bites became one of mine.**

I wanted to practice:

* Designing visually appealing layouts
* Creating responsive sections
* Organizing project files professionally
* Writing cleaner HTML and CSS
* Adding interactive behavior with JavaScript
* Using Git and GitHub like a real developer

This project reminded me that **growth happens when you move from consuming knowledge to applying it**.

---

## 🎯 Project Overview

**Tasty Bites** is a fictional restaurant website designed to simulate a real-world landing page experience.

The focus of the project is not only aesthetics, but also:

* Semantic HTML structure
* Responsive web design
* User-friendly navigation
* Interactive UI elements
* Clean code organization
* Maintainable frontend architecture
